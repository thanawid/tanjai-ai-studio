# AI Handoff Prompt

ใช้คำสั่งนี้กับ Cursor / Claude Code / Codex ก่อนให้แก้เว็บ

คุณกำลังแก้เว็บ “ทันใจ AI Studio”
เว็บนี้เป็น Prompt Workspace สำหรับช่วยสร้างบรีฟและ Prompt Pack สำหรับผลิตสื่อประชาสัมพันธ์ / คอนเทนต์โซเชียล / ชุดภาพโพสต์ Facebook

กติกา:
- อ่าน PROJECT_RULES.md ก่อนแก้ทุกครั้ง
- แก้เฉพาะจุดที่สั่ง
- ห้ามรื้อ layout เดิมโดยไม่จำเป็น
- ตรวจ Desktop / Tablet / Mobile ทุกครั้ง
- คง ZIP / Caption / Preview / Sidebar / Hero เว้นแต่ผู้ใช้สั่ง
- ห้ามใส่เลขลำดับบนภาพจริง
- ห้ามใส่ hashtag บนภาพจริง
- ภาพปกและภาพรองในโมดูล Album ต้องแยกกัน
- Preview กับ Output ต้องตรงกันมากที่สุด
- ก่อนส่งงานให้สรุปว่าแก้อะไร ไม่แตะอะไร และไฟล์ใดเปลี่ยน


แนวทาง Universal Prompt Pack:
- ผู้ใช้กรอกข้อมูลครั้งเดียว
- ระบบสร้าง Prompt กลางและ Prompt แยกตามประเภทงาน
- ห้ามบังคับเลือก AI ปลายทางก่อน
