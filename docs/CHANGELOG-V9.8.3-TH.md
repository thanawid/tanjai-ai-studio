# Tanjai AI Studio V9.8.3 — Stable Foundation

สถานะ: แพตช์จัดฐานให้พร้อมใช้งานจริงมากขึ้น โดยไม่เปลี่ยนโครงหลักของเว็บ

## สิ่งที่ปรับในเวอร์ชันนี้

1. จัดเลขเวอร์ชันให้ตรงทั้งระบบ
   - Badge หน้า Login
   - Badge บน Dashboard
   - คู่มือระบบ
   - Cache bust ของ CSS/JS
   - package.json

2. ซ่อน AI เฉพาะทางออกจากผู้ใช้
   - ผู้ใช้ไม่ต้องเลือก “ผู้เชี่ยวชาญ AI” เอง
   - ระบบเลือกบทบาทหลังบ้านจากแนวภาพ ประเภทงาน หัวข้อ และรายละเอียด
   - ยังส่งบทบาท AI เฉพาะทางเข้า Prompt เพื่อให้ผลลัพธ์แม่นขึ้น

3. ปรับ Smart Thai Proofread ให้ใช้จริงขึ้น
   - มีปุ่มตรวจคำผิดด้านบน
   - เพิ่มปุ่มตรวจคำผิดในเมนูซ้าย
   - ตรวจอัตโนมัติก่อนกดสร้างงานหลัก
   - เพิ่มคำผิด/คำมาตรฐานที่เจอบ่อยในงานประชาสัมพันธ์และราชการ
   - เพิ่มการจัดรูปแบบ พ.ศ. วัน เวลา และ QR Code เบื้องต้น
   - ยังแก้เฉพาะคำที่ระบบมั่นใจ และให้ผู้ใช้เพิ่มคำสำคัญขององค์กรเพื่อกันชื่อเฉพาะ

4. ปรับคู่มือให้เป็นแบบสั้น ใช้จริงในทีม
   - เน้นวิธีเลือกเมนู กรอกข้อมูล ตรวจคำผิด และคัดลอกไปใช้ต่อ
   - ไม่ทำเป็นคู่มือยาวเกินจำเป็น

5. ปรับกรอบแผนบริการแบบ MVP
   - ทดลองฟรี 14 วัน
   - Pro 490 บาท/เดือน/ผู้ใช้
   - ทีม/องค์กรเริ่มต้น 1,490 บาท/เดือน สำหรับ 3–5 ผู้ใช้
   - ระบุว่าเป็นกรอบทดสอบตลาด ยังไม่รวมต้นทุน API หากต่อยอดระบบ AI เต็มรูปแบบ

6. ลดจำนวนไฟล์
   - ตัด asset ตัวอย่างที่ไม่ได้ถูกอ้างอิงออก
   - คงไฟล์ preview ที่หน้าเว็บเรียกใช้จริง

## หมายเหตุสำคัญ

เวอร์ชันนี้ยังเป็นระบบช่วยจัดบรีฟและสร้าง Prompt/ข้อความพร้อมใช้ ไม่ใช่ระบบสร้างภาพ/เสียงผ่าน API โดยตรง ต้นทุน API และระบบชำระเงินควรประเมินอีกครั้งก่อนเปิดขายจริงเต็มรูปแบบ
