# TANJAI AI Studio — V9.8.3

อัปเดตทดลอง: PR Recap + Specialist Split

## เพิ่มในเมนูสร้างภาพ
- เพิ่มช่อง “ผู้เชี่ยวชาญ AI” เพื่อแยกบทบาทงานภาพให้ชัด
- เพิ่ม preset แนวภาพ:
  - ข่าวประชาสัมพันธ์แบบสรุปกิจกรรม
  - โปสเตอร์สถาบันการศึกษา
  - PR Recap / Photo Grid Poster
- เพิ่ม quick chips สำหรับงานข่าวสรุปกิจกรรม / สถาบันการศึกษา / Photo Grid PR

## Specialist ใหม่
- AI Poster Creative Director
- AI PR News Designer
- AI Event Recap Designer
- AI Education Poster Designer
- AI Infographic Planner
- AI Character Visual Designer
- AI Cinematic Cover Director

## ปรับ Prompt Engine
- แทรกบทบาท Specialist ลงใน Prompt ภาพโดยตรง
- แยก “สไตล์ภาพ” ออกจาก “ข้อความบนภาพ” ชัดขึ้น
- เพิ่ม logic งาน PR recap / education poster ให้จัดหัวภาพ + ภาพหลัก + กริดภาพย่อย + บล็อกข้อมูล
- เปลี่ยนผลลัพธ์หน้า Output ให้แสดงชื่อตาม Specialist ที่ระบบเลือก

## หมายเหตุ
- อัปเดตนี้เน้นทดลองแนวงานแบบตัวอย่างภาพข่าวประชาสัมพันธ์หลายส่วน
- โครงสร้างอื่นของเว็บพยายามคงเดิมให้มากที่สุด
