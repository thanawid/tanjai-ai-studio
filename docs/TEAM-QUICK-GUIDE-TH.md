# คู่มือใช้งานทีมแบบสั้น — ทันใจ AI Studio

## ใช้ยังไงให้เร็ว

1. เลือกเมนูตามผลลัพธ์ที่ต้องการ เช่น สร้างภาพ เรียบเรียงเนื้อหา ทำวิดีโอ เสียงพากย์ หรือทำสไลด์
2. กรอกข้อมูลจริงเท่าที่มี โดยเฉพาะหัวข้อ หน่วยงาน วันเวลา สถานที่ และบุคคลสำคัญ
3. ใส่สิ่งที่ห้าม AI แต่งเพิ่ม เช่น QR Code เบอร์โทร ราคา ค่าสมัคร วันที่ หรือชื่อตำแหน่ง
4. กดตรวจคำผิดเมื่อเป็นงานทางการ หรือเพิ่มคำสำคัญขององค์กรเพื่อกันชื่อเฉพาะ
5. กดสร้าง แล้วคัดลอกผลลัพธ์ไปใช้ต่อใน ChatGPT, Canva, CapCut หรือเครื่องมือปลายทาง

## หลักใช้งานสำหรับทีม

- คนกรอกข้อมูลต้องใส่ข้อมูลจริงให้ครบที่สุด
- คนตรวจงานต้องตรวจชื่อคน หน่วยงาน ตำแหน่ง วันเวลา และสถานที่ก่อนเผยแพร่
- ถ้าเป็นภาพจริง ให้แนบภาพต้นฉบับไปพร้อม Prompt ทุกครั้ง
- ถ้าเป็นงานราชการหรือประชาสัมพันธ์ ให้ใช้ภาษาสุภาพ ชัดเจน และไม่โฆษณาเกินจริง
- ระบบช่วยจัดงานและลดเวลาคิด ไม่ใช่ผู้รับรองความถูกต้องของข้อเท็จจริง
