# V9.3.13 — Facebook Publishing Director

ปรับเฉพาะเมนู **ชุดภาพโพสต์ Facebook** ต่อจาก V9.3.12 รวม 5 เรื่อง

1. ปรับกรอบ Cover / Lite ให้เป็นกรอบ Civic แบบสะอาด ลดเส้นโค้งและสีซ้อนที่ดูผิดรูป
2. ย่อส่วน “ตรวจภาพรายใบและดาวน์โหลดแยก” พร้อมจำกัดความสูงและขนาดภาพเพิ่มเติม
3. เปลี่ยนการเลือก Layout Facebook จาก Dropdown เป็นการ์ดภาพที่กดเลือกได้ทันที
4. แยก `Caption Writer` และ `Fact Guard` ออกจากระบบภาพ พร้อมสไตล์แคปชั่น 5 แบบ
5. ปรับช่องจัดลำดับภาพรองให้เต็มความกว้าง อ่านชื่อไฟล์ง่าย และใช้ลูกศรซ้าย–ขวา

ระบบภาพและแคปชั่นใช้ข้อมูลจริงชุดเดียวกัน แต่ประมวลผลด้วยโมดูลเฉพาะทางคนละหน้าที่
