# V9.1 — Context & Creative Quality System

เป้าหมาย:
ให้เว็บช่วยคิดเชิงสร้างสรรค์แบบพอดีตามบริบทงาน แต่ไม่มั่วข้อมูลจริง ไม่เว่อร์ และปล่อย Prompt ที่เอาไปสร้างงานต่อได้จริง

## สิ่งที่เพิ่ม
1. บริบทงาน
- ให้ AI ช่วยเลือกจากรายละเอียด
- แจ้งข่าว / ประกาศ
- เชิญชวน / ประชาสัมพันธ์
- โปรโมท / แคมเปญ / ขายผลงาน
- โปรโมทเพลง / ผลงานสร้างสรรค์
- ให้ความรู้ / Infographic
- ขั้นตอนบริการ / วิธีใช้งาน
- สรุปกิจกรรม / รายงานผล
- งานพิธี / งานบุญ / งานชุมชน
- อบรม / ประชุม / สัมมนา
- ขอบคุณ / แสดงความยินดี / อวยพร
- รณรงค์ / สร้างความตระหนัก
- ไว้อาลัย / สุภาพ / ลดสี
- อื่น ๆ

2. ประเภทภาพ
- โพสต์โซเชียล
- โปสเตอร์ประชาสัมพันธ์
- อินโฟกราฟิก
- ภาพแน่นข้อมูล
- ภาพอ่านง่าย
- ปก / Cover
- ปกเพลง / โปรโมทเพลง
- ภาพเชิญชวนกิจกรรม
- ภาพแจ้งข่าว
- ภาพสรุปกิจกรรม
- ชุดภาพโพสต์ Facebook
- ภาพแนวนอน / แบนเนอร์
- ภาพไว้อาลัย / สุภาพ
- อื่น ๆ

3. Creative Quality Lock
- งานเพลง = อารมณ์เพลง / ชื่อเพลงเด่น / ไม่แน่นแบบ infographic
- แจ้งข่าว = อ่านเร็ว / หัวข้อชัด / ไม่ตกแต่งเกิน
- เชิญชวน = ชื่องาน + บรรยากาศ + กล่องข้อมูลถ้ามีจริง
- ให้ความรู้ = ข้อมูลเป็นลำดับ / infographic อ่านง่าย
- สรุปกิจกรรม = ภาพจริงและบรรยากาศเด่น
- ไว้อาลัย = ลดสี / สุภาพ / ไม่ฉูดฉาด

## หลักระบบ
- ผู้ใช้กรอกน้อย
- เว็บช่วยคิดหมวด สรุป และเลือกโทนให้
- ระบบคิดต่อได้ แต่ห้ามแต่งข้อมูลจริงเอง
- Prompt สุดท้ายต้องเป็นคำสั่งพร้อมใช้ ไม่ใช่บรีฟเลอะ ๆ
- แต่ละเมนูยังใช้ logic ของตัวเอง
