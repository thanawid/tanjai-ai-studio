1เลือกงานภาพ / โพสต์ / วิดีโอ / เสียง / สไลด์

3คัดลอก Promptนำไปใช้กับ ChatGPT, Canva, CapCut หรือเครื่องมือ AI ที่ต้องการ

🧭 AI Router

บอกระบบว่าพี่อยากทำอะไร แล้วระบบจะแนะนำเมนู ผลลัพธ์ และเครื่องมือปลายทางให้

🖼️ สร้างภาพ

ผู้กำกับงานภาพ: คุมสาร ลำดับสายตา ข้อความไทย ภาพจริง และ Prompt พร้อมผลิต

✨ AI Photo Pro

แต่งภาพจริงหลายภาพแบบอัตโนมัติ ปรับแสง สี เงา และความคมชัด โดยไม่สร้างภาพใหม่

🧷 ชุดภาพโพสต์ Facebook

อัปโหลดหลายภาพ ปรับแสงสี ครอป ใส่กรอบข้อความ และดาวน์โหลดภาพพร้อมโพสต์ โดยไม่สร้างภาพใหม่

✍️ เรียบเรียงเนื้อหา

ผู้เชี่ยวชาญงานเขียนสำหรับสรุปงาน ข่าวประชาสัมพันธ์ โพสต์ Facebook ข้อความ Line และแคปชั่นโดยเฉพาะ

🎤 งานพิธีกร

ผู้กำกับงานเวที: คุมลำดับพิธี ชื่อ ตำแหน่ง คำเชื่อม คำกำกับเวที และบัตรคำพูด

🎬 ทำวิดีโอ

ผู้กำกับวิดีโอ: คุมประโยคเปิด เส้นเรื่อง เวลารวม ลำดับภาพ บทพากย์ รายการช็อต และจังหวะตัดต่อ

🎙️ เสียงพากย์

ผู้กำกับเสียง: เขียนเพื่อการฟัง คุมเวลา จังหวะ คำเน้น อารมณ์ และคำอ่านชื่อเฉพาะ

📊 ทำสไลด์

นักวางโครงนำเสนอ: คุมเส้นเรื่อง หัวข้อสรุป หลักฐานภาพ และบันทึกผู้พูด

🧩 สร้างชุดสื่อ

ผู้กำกับชุดสื่อ: สร้างแกนกลางหนึ่งชุด แล้วแยกบทบาทของแต่ละสื่อให้สอดคล้องโดยไม่คัดลอกกัน

⭐ คลัง Prompt

คลังคำสั่งพร้อมใช้สำหรับงานสื่อสารองค์กร โซเชียล และแบรนด์

🚀 เครื่องมือปลายทาง

รวมเครื่องมือที่ใช้ต่อหลังได้ Prompt เช่น ทันใจ GPT, Canva, CapCut และ Voice Tool

📁 โปรเจกต์

เก็บผลลัพธ์ที่ใช้งานบ่อยไว้กลับมาใช้ต่อภายหลัง

⭐ ตัวอย่างงานพร้อมใช้

เลือกแนวงานที่ใกล้เคียง แล้วระบบจะพาไปกรอกข้อมูลต่อในเมนูที่เหมาะสมทันที

ตัวอย่างสำหรับขาย MVP: เทศบาล / อบต. โรงเรียน เพจ / ครีเอเตอร์ ร้านค้า

⚙️ คู่มือระบบ

01

เลือกประเภทงาน

เช่น ภาพประชาสัมพันธ์ โพสต์ Facebook วิดีโอ เสียงพากย์ หรือสไลด์

02

กรอกข้อมูลจริง

ใส่หัวข้อ หน่วยงาน รายละเอียด วันเวลา สถานที่ และบุคคลสำคัญเท่าที่มี

03

กดสร้าง Prompt

ระบบจะแยก Prompt สำหรับแก้/คุยต่อ และ Prompt สำหรับสั่ง AI ทำงานทันที

04

คัดลอกไปใช้ต่อ

นำไปวางใน ChatGPT, Canva, CapCut หรือเครื่องมือปลายทางที่ต้องการ แล้วแนบรูปประกอบถ้ามี

จำง่าย ๆ

  1. เลือกงาน
  2. กรอกข้อมูล
  3. กดสร้าง
  4. คัดลอก Prompt
  5. นำไปใช้กับ AI ปลายทาง

คำเตือนสำคัญ

  1. ห้ามแต่งข้อมูลจริงเพิ่มเอง
  2. ตรวจชื่อคน หน่วยงาน วันที่ และสถานที่ก่อนเผยแพร่
  3. ถ้า AI ปลายทางสร้างภาพไม่ได้ ให้ใช้ Prompt ที่ระบบเตรียมไว้ไปวางในเครื่องมือสร้างภาพ
  4. ถ้ามีรูปถ่าย ให้แนบรูปไปพร้อม Prompt ทุกครั้ง
