# V9.3.2 — Balanced Creative + Attachment Handoff

ฐาน: V9.3.1 Thai PR Premium

สิ่งที่แก้:
- เปลี่ยน Smart Default เป็น `Balanced Creative PR` — มืออาชีพ มี Big Idea และครีเอทีฟพอดี โดยไม่บังคับ 3D หรือข้อมูลแน่น
- แยกเส้นทางอัตโนมัติ 4 แบบ: Balanced Creative, Thai PR Premium, Photo-led และ Respectful Editorial
- ใช้ Thai PR Premium แบบตัวอย่างเฉพาะเมื่อผู้ใช้เลือกหรือบรีฟเป็นงานประชาสัมพันธ์ข้อมูลแน่นที่เหมาะจริง
- เปลี่ยนค่าเริ่มต้นระดับการคิดเป็น “คิดสร้างสรรค์ระดับกลาง” แต่ยังล็อกข้อเท็จจริงทุกชนิด
- แยกปุ่มเมนูภาพเป็น `1. คัดลอก Prompt` และ `2. เปิด ทันใจ GPT`
- ยกเลิกข้อความที่ทำให้เข้าใจว่าการเปิด GPT จะส่งไฟล์แนบตามไปอัตโนมัติ
- แสดงกล่องเตือนรายชื่อไฟล์จริงที่ผู้ใช้ต้องกด `+` แนบซ้ำใน GPT
- เพิ่ม `ATTACHMENT HANDOFF CHECK` ใน Prompt เพื่อแยกกรณี “ยังไม่ได้แนบไฟล์” ออกจาก “เครื่องมือสร้างภาพใช้ไม่ได้”
- เมื่อไม่พบไฟล์จริงในแชท GPT ต้องขอให้ผู้ใช้แนบไฟล์ก่อน ห้ามสร้างต่อจากชื่อไฟล์เพียงอย่างเดียว
- คง Thai PR Premium Mode, Album Frame System และระบบเมนูเดิมทั้งหมด

ข้อจำกัดจากเบราว์เซอร์:
- เว็บไซต์ GitHub Pages ไม่สามารถส่งไฟล์จาก `<input type="file">` ไปยัง chatgpt.com อัตโนมัติได้
- Custom GPT ไม่มี API ฟรีสำหรับฝังผลลัพธ์กลับมาในเว็บไซต์โดยตรง
