# V9.7 — AI Photo Pro Basic Batch Enhancer

ฐาน: ไฟล์ `tanjai-ai-studio-main(1).zip` ที่ผู้ใช้อัปโหลด

ขอบเขต:
- เพิ่มเมนูใหม่ต่อจาก “สร้างภาพ” คือ **AI Photo Pro / แต่งภาพ AI**
- เมนูอื่นคงเดิม
- ทำเป็นระบบปรับภาพจริง ไม่ใช่ระบบสร้างภาพใหม่

ฟีเจอร์ใหม่:
- อัปโหลดรูปหลายภาพพร้อมกัน
- ปรับแสง สี เงา ความคมชัดในเว็บด้วย Canvas
- Preset: Auto Pro, ภาพประชุม/กิจกรรม, ภาพในห้องแสงน้อย, กลางแจ้ง, ภาพบุคคล, สินค้า/อาหาร, แก้ภาพมืด, สุภาพ/ราชการ, เพิ่มความคม
- ปรับละเอียดด้วย slider: brightness, contrast, saturation, warmth, shadows, highlights, sharpness
- จำกัดขนาดด้านยาว 2048/1600/1080 หรือคงขนาดต้นฉบับ
- ดาวน์โหลดทีละภาพหรือรวมเป็น ZIP
- Safe Mode: ไม่เปลี่ยนใบหน้า ไม่สร้างวัตถุ ไม่สร้างฉากใหม่

หมายเหตุ:
- เป็น Basic Batch Enhancer ที่ประมวลผลในเครื่องผู้ใช้ ไม่ส่งภาพออกไป API
- ยังไม่ใช่ AI Pro API สำหรับ denoise/upscale/deblur ระดับ server
- แพ็ก ZIP ลดจำนวนไฟล์ให้ต่ำกว่า 100 เพื่ออัป GitHub ได้ง่ายขึ้น

- JavaScript syntax ผ่านทุกไฟล์หลัก
- Version badge เป็น V9.7
