# V9.1.6 — Image Menu Simplified Workflow

อัปเดตเฉพาะเมนู **สร้างภาพ** โดยคงระบบอื่นเดิมทั้งหมด

## สิ่งที่ปรับ
- ลดโครงหน้าเมนูสร้างภาพจากหลายช่วงย่อย ให้เหลือ **3 ขั้นตอนหลัก**
  1. ข้อมูลงาน
  2. รูปแบบภาพ
  3. ตั้งค่าเพิ่มเติม
- เพิ่มช่อง **ข้อความบนภาพ (ถ้ามี)** เพื่อให้คัดลอก Prompt ไปใช้จริงได้ตรงขึ้น
- ยุบตัวเลือกที่ไม่จำเป็นไว้ใน **ตัวเลือกขั้นสูง** ลดความรกหน้าเว็บ
- เพิ่มปุ่มลัด **เทศบาล ม่วง–ทอง** และตั้งค่าโทนเทศบาลอัตโนมัติเมื่อพบคำว่าเทศบาล / อบต. / หน่วยงานราชการ
- คงระบบปกป้องภาพจริงและป้องกันหน้าเพี้ยนไว้ แต่ย้ายไปอยู่ในส่วนที่ไม่รบกวนผู้ใช้ทั่วไป
- ปรับ layout ของเมนูสร้างภาพให้รองรับ **Desktop / Tablet / Mobile** ดีขึ้น
