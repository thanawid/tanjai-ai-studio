# แผนแสดงผลในเว็บแบบฟรีก่อน

## ข้อเท็จจริงทางเทคนิค

- GitHub Pages เป็นเว็บแบบ static จึงเรียก Custom GPT และอ่านผลลัพธ์จากแชทกลับมาแสดงในเว็บไซต์ไม่ได้
- Custom GPT ไม่มี API ฟรีสำหรับฝังในเว็บไซต์
- การเรียก OpenAI API จากเว็บโดยตรงต้องเสียค่าใช้งานและห้ามฝังคีย์ลับไว้ใน JavaScript ฝั่งผู้ใช้
- บริการสร้างภาพฟรีภายนอกมักมีโควตา ความเสถียร และความเป็นส่วนตัวที่ควบคุมไม่ได้ จึงไม่เหมาะกับภาพบุคคลหรือเอกสารหน่วยงาน

## ทางฟรีที่แนะนำ

เพิ่ม **Local Preview Mode** ในเว็บ:

1. ผู้ใช้กรอกข้อความและเลือกภาพจริง
2. เว็บวางข้อความ กรอบ สี โลโก้จริง และภาพจริงด้วย Canvas ภายในเครื่อง
3. แสดง Preview และดาวน์โหลด PNG/JPG ได้ทันที
4. ไม่ส่งข้อมูลออกจากอุปกรณ์และไม่มีค่า API

ข้อจำกัด: เป็นระบบ Template/Canvas จึงไม่สามารถวาดฉากหรือภาพประกอบใหม่ระดับ Generative AI ได้ แต่เหมาะสำหรับโปสเตอร์พื้นฐาน Quote Card กรอบข่าว และภาพประชาสัมพันธ์จากรูปจริง

## ทางเต็มในอนาคต

หากต้องการให้เว็บสร้างภาพ AI และแสดงผลในหน้าเว็บโดยตรงอย่างสม่ำเสมอ ต้องมี backend สำหรับเก็บ API key, จัดการไฟล์แนบ, เรียก Image API และส่งผลลัพธ์กลับมา ซึ่งมีค่าใช้จ่ายตามการใช้งาน

## ลำดับแนะนำ

- ตอนนี้: ใช้ 2 ปุ่ม `คัดลอก Prompt` + `เปิด ทันใจ GPT` และแนบไฟล์เอง
- รุ่นถัดไปแบบฟรี: Local Preview Mode
- เมื่อพร้อมเก็บเงิน: Backend + Image API + ระบบเครดิต/แพ็กเกจ
