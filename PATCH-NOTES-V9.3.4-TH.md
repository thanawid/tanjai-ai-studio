# V9.3.4 — Smart Facebook Cover & Preview

## สิ่งที่แก้

- ปรับภาพแรกให้เป็น Cover Frame สำหรับขายหัวข้อจริง: แถบหน่วยงานด้านบน หัวข้อหลักเด่น และแถบข้อมูลด้านล่าง โดยลดกล่องทึบที่บังภาพ
- ยกเลิกหมายเลข 2–3–4–5 บนภาพรองและภาพเพิ่มเติมทั้งหมด
- เพิ่มขนาด `อัตโนมัติ` เป็นค่าเริ่มต้น
  - ภาพปกแนวนอน → 16:9
  - ภาพปกแนวตั้ง → 4:5
  - ภาพปกใกล้จัตุรัส → 1:1
- เพิ่มรูปแบบตัวอย่างโพสต์ 3 แบบ
  - ปกใหญ่ด้านบน + ภาพรองด้านล่าง
  - ปกใหญ่ด้านซ้าย + ภาพรองด้านขวา
  - ตารางภาพสมดุล
- โหมดพรีวิวอัตโนมัติเลือกจากสัดส่วนและจำนวนภาพ
- พรีวิว Facebook ใช้รูปแบบที่ผู้ใช้เลือกหรือระบบตัดสินใจจริง พร้อมแสดงสัดส่วนและชื่อ Layout
- เมื่อเปลี่ยนขนาดหลังสร้าง ระบบสร้างชุดภาพใหม่ให้ตรงกับขนาดที่เลือก

## หลักการ

ภาพแรกทำหน้าที่ขายเรื่อง ส่วนภาพต่อไปทำหน้าที่ขยายบรรยากาศ จึงไม่ใส่เลขลำดับที่ทำให้งานดูเหมือนสไลด์นำเสนอ
