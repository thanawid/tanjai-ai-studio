# Universal Prompt Pack Spec

## เป้าหมาย
ผู้ใช้ไม่ต้องเลือก AI ปลายทางก่อน

Flow หลัก:
1. เลือกประเภทงานหรือเมนู
2. กรอกข้อมูลจริงของงาน
3. กดสร้าง
4. ได้ Universal Prompt Pack พร้อมใช้ต่อกับ AI หลายตัว

## Universal Prompt Pack ต้องมี
1. Prompt กลางพร้อมใช้ทุก AI
2. Smart Brief
3. Prompt สร้างภาพประชาสัมพันธ์
4. Prompt ชุดภาพ Facebook
5. Caption พร้อมโพสต์
6. Prompt วิดีโอ / Storyboard
7. สคริปต์เสียงบรรยาย
8. Outline สไลด์
9. ข้อความบนภาพ / ปกคลิป
10. คำสำคัญที่ต้องสะกดตรง
11. วิธีใช้ต่อกับ AI หลายตัว
12. ข้อกำชับกันหลุด

## หลักการ
- ไม่บังคับผู้ใช้เลือก ChatGPT / Gemini / Claude / Canva / CapCut / Runway / Kling ก่อน
- ระบบต้องสร้าง Prompt แบบกลางที่ AI ส่วนใหญ่เข้าใจได้
- งานภาพต้องได้ Prompt ภาพ
- งานวิดีโอต้องได้ Prompt วิดีโอ / storyboard
- งานเสียงต้องได้สคริปต์เสียง
- งานสไลด์ต้องได้ outline และ speaker notes
- ชื่อบุคคล หน่วยงาน สถานที่ วันที่ ต้องคงตามผู้ใช้
