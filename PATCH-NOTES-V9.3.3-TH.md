# V9.3.3 — Facebook Album Result Hierarchy

ปรับเมนูชุดภาพโพสต์ Facebook ให้ตัดสินใจและใช้งานเร็วขึ้น:

- แสดง `ตัวอย่างโพสต์ Facebook` เป็นผลลัพธ์แรกทางด้านขวา
- แสดงสถานะจำนวนภาพ พร้อมปุ่มคัดลอกแคปชั่นและดาวน์โหลด ZIP ใกล้ตัวอย่างโพสต์
- ย้ายตัวแก้ไขแคปชั่นไว้ในส่วนพับได้ และอัปเดตตัวอย่างโพสต์ทันทีขณะพิมพ์
- รวมภาพปก ภาพรอง และภาพเพิ่มเติมไว้ใน `ตรวจภาพรายใบและดาวน์โหลดแยก`
- ใช้ชื่อหัวข้อภาษาไทยเป็นหลัก ลดคำอังกฤษที่ไม่จำเป็น
- ลดผลลัพธ์ซ้ำและลดระยะเลื่อน โดยยังเก็บการดาวน์โหลดรายภาพครบ

ลำดับผลลัพธ์ใหม่:

1. ตัวอย่างโพสต์จริง
2. แก้ไขแคปชั่น
3. ตรวจภาพรายใบและดาวน์โหลดแยก
