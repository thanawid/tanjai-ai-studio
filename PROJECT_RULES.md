# ทันใจ AI Studio — Project Rules

## หลักการใหญ่
เว็บนี้คือ Prompt Workspace / Media Brief Assistant สำหรับช่วยผู้ใช้กรอกข้อมูลครั้งเดียว แล้วได้ชุดคำสั่งพร้อมนำไปใช้ต่อกับ AI ตัวอื่น เช่น ChatGPT, Gemini, Claude, Canva, CapCut, Runway, Kling

## กติกาการแก้ไข
1. แก้เฉพาะจุดที่สั่ง
2. อย่างอื่นคงเดิมทั้งหมด
3. ห้ามเพิ่มฟีเจอร์เองโดยไม่สรุปก่อน
4. ทุกการแก้ต้องคิด Desktop / Tablet / Mobile พร้อมกัน
5. ห้ามทำให้ Sidebar / Bottom nav / Floating button ทับกัน
6. Preview และ Output ต้องใช้ logic เดียวกันเท่าที่ทำได้
7. ZIP / Caption / Preview ต้องยังใช้งานได้
8. ห้ามเปลี่ยน Hero / Sidebar / เมนูอื่น ถ้าไม่ได้สั่ง
9. ก่อนสร้าง ZIP ต้องสรุปว่าแก้อะไรและไม่แตะอะไร
10. ถ้าไม่เข้าใจให้ถามก่อน ห้ามเดา

## ตัวตนของระบบ
- ไม่จำเป็นต้องเจนภาพเองทั้งหมดในระยะนี้
- หน้าที่หลักคือจัดบรีฟ สร้าง Prompt Pack และเตรียมข้อมูลให้ AI ตัวอื่นทำงานได้ตรงใจ
- ต้องลดการโดดไปโดดมา โดยมีคำสั่งกลางและกติกาการส่งต่อ AI


## V8.7.4 Universal Prompt Pack
- ไม่บังคับผู้ใช้เลือก AI ปลายทางก่อน
- ระบบต้องสร้าง Prompt พร้อมใช้ต่อได้หลาย AI จากข้อมูลเดียว
- ตัวเลือก AI ปลายทางเป็นเรื่องรอง ไม่ใช่ขั้นตอนหลัก
