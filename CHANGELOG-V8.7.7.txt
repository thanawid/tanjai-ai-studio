V8.7.7 — MVP Sales Polish

เป้าหมาย: ทำให้เว็บพร้อมให้คนอื่นเข้าใจและทดลองใช้จริงมากขึ้น หลังจาก V8.7.7 อัปขึ้นเว็บแล้ว

สิ่งที่ปรับ:
- ปรับข้อความหน้าแรกให้สื่อสารชัดว่าเป็นระบบช่วยสร้าง Prompt และชุดคำสั่งผลิตสื่อ
- เพิ่มบล็อก “เว็บนี้ช่วยอะไร?” พร้อมขั้นตอนสั้น 3 ขั้น
- เพิ่มหน้า “ตัวอย่างงานพร้อมใช้” สำหรับเทศบาล/อบต., งานลงพื้นที่, โรงเรียน/ศูนย์เด็กเล็ก, เพจ/ครีเอเตอร์ และร้านค้า
- ตัวอย่างงานมีปุ่ม “ใช้ตัวอย่างนี้” และ “สร้าง Prompt ครบชุด”
- เพิ่มคู่มือ “ใช้งาน 3 นาทีเป็น” สำหรับผู้ใช้ใหม่
- ปรับ Responsive เพิ่มเติมสำหรับ Desktop / Tablet / Mobile
- ลดความแน่นของปุ่มและการ์ดบนมือถือ
- คง Login, Sidebar, Hero โครงเดิม, Prompt Pack, Album, ZIP, Caption, Preview และ FAB V8.7.7

หมายเหตุ:
- ยังเป็น Static Web App / MVP
- ระบบ Login เดิมยังเป็น Front Gate ไม่ใช่ระบบสมาชิกจริง
- การต่อ AI API / เก็บงานข้ามเครื่อง / Login จริง ควรทำในสาย V8.8.x
