# Responsive Checklist

ทุกเวอร์ชันต้องตรวจ:

## Desktop
- Sidebar ใช้งานได้
- Result panel ไม่ล้น
- Preview อ่านง่าย
- ปุ่มหลักไม่ชนกัน

## Tablet
- ฟอร์มไม่แน่นเกิน
- Preview ไม่แคบจนอ่านข้อความไม่ได้
- ปุ่มยังแตะได้ง่าย

## Mobile
- ไม่โชว์ปุ่มเยอะเกิน
- Bottom nav ไม่ชน floating button
- Version badge ไม่เด่นเกินงานหลัก
- ฟอร์มเรียงอ่านง่าย
- Preview / caption / copy button ใช้งานได้

## Output
- ZIP ดาวน์โหลดได้
- Caption คัดลอกได้
- Prompt Pack คัดลอกได้
- ข้อความไม่ล้น
