# ทันใจ AI Studio v6.1 — Tool Forms Pro

- เพิ่มคำว่า “ทำคลิป” ให้ชัดในเมนูวิดีโอ

- เพิ่ม dropdown เฉพาะงานให้โพสต์ วิดีโอ เสียงพากย์ และสไลด์
