# ทันใจ AI Studio v8.4.1 — Clean Ready Output Pro

เวอร์ชันนี้ปรับโครงผลลัพธ์ตามแนว **A + B** ที่คุยกัน โดยเน้นให้ “อ่านง่าย ใช้งานจริงเร็ว และไม่รก”

## สิ่งที่ปรับในเวอร์ชันนี้
- เปลี่ยนโครงผลลัพธ์ของเมนูหลักเป็น **ผลลัพธ์พร้อมใช้ก่อน**
- แยก **รายละเอียดเพิ่มเติม** ออกไปไว้ในส่วนพับเก็บ (Advanced)
- หน้า “สร้างภาพ” แสดง **Prompt พร้อมใช้** เป็นผลลัพธ์หลัก
- ซ่อน **คำสั่งพร้อมใช้สำหรับทันใจ GPT** และ **Auto Prompt Critic** ไว้ในส่วนรายละเอียดเพิ่มเติม
- ลดความรกของปุ่มบนฝั่งขวา เหลือเฉพาะ **ปุ่มหลักที่ควรกดก่อน**
- ย้ายลิงก์เครื่องมืออื่น ๆ ไปไว้ในส่วน **เครื่องมือที่แนะนำ** ด้านล่าง
- ปรับชื่อผลลัพธ์ให้สื่อความหมายชัดขึ้น เช่น
  - Prompt พร้อมใช้
  - ข้อความพร้อมใช้
  - Storyboard พร้อมใช้
  - สคริปต์เสียงพร้อมใช้
  - Outline พร้อมใช้
  - ชุดสื่อพร้อมใช้

## ผลลัพธ์ที่ได้
ผู้ใช้จะเห็นลำดับแบบนี้:
1. สิ่งที่คัดลอกไปใช้ได้ทันที
2. ปุ่มหลักที่ควรกดก่อน
3. รายละเอียดเพิ่มเติมเมื่อจำเป็น

## วิธีอัปเดตบน GitHub Pages
1. แตก ZIP นี้
2. อัปโหลด **ทุกไฟล์และทุกโฟลเดอร์** ไปทับของเดิมใน repo
3. Commit changes
4. รอ GitHub Pages deploy
5. เข้าเว็บแล้วกด Refresh แบบ hard reload 1 ครั้ง

## V8.4.1 — Natural Thai Voice Script

- ปรับเฉพาะเมนู “เสียงพากย์”
- สคริปต์เสียงออกมาเป็นภาษาประกาศข่าวท้องถิ่นที่อ่านจริงได้
- จัดจังหวะเป็นย่อหน้า: เปิดประกาศ / วันที่ / ผู้เกี่ยวข้อง / ภารกิจ / สถานที่ / วัตถุประสงค์ / ปิดท้ายองค์กร
- ลดภาษาฟอร์ม เช่น “วันและเวลา:” หรือ “สถานที่:” ที่แข็งเกินไป
- เหมาะกับงานเทศบาล หน่วยงานราชการ คลิปสรุปงาน และเสียงพากย์ประชาสัมพันธ์
- อย่างอื่นคงเดิมจาก V8.3


## V8.4.1 — Facebook Album Auto Frame & Smart Enhance

เพิ่มเมนูใหม่ **ชุดภาพโพสต์ Facebook**

- อัปโหลดภาพจริงหลายภาพ
- ครอปอัตโนมัติเป็น 4:5 / 1:1 / 16:9 / 9:16
- ปรับแสง สี คอนทราสต์ และความคมชัดแบบเบา ๆ ด้วย Canvas
- ใส่กรอบข้อความในภาพแบบไม่สร้างภาพใหม่
- รูปที่ 1 เป็นปกหลัก
- รูปที่ 2 เป็นปกรอง / สรุปข้อมูล
- รูปที่ 3 เป็นต้นไปเป็นภาพจริงพร้อมกรอบล่าง
- ดาวน์โหลดทีละภาพ หรือดาวน์โหลดทั้งหมด
- มี Safe Photo Mode: ไม่สร้างภาพใหม่ ไม่เปลี่ยนใบหน้า ไม่แก้องค์ประกอบหลัก


## V8.4.1 — Album ZIP Download Fix

- แก้ปุ่ม **ดาวน์โหลดทั้งหมด** ให้โหลดเป็นไฟล์ ZIP ไฟล์เดียว
- ยังสามารถดาวน์โหลดทีละภาพจากพรีวิวได้เหมือนเดิม
- ใช้ ZIP generator ในเบราว์เซอร์ ไม่ต้องต่อ server และไม่ต้องใช้ library ภายนอก
- อย่างอื่นคงเดิมจาก V8.4
