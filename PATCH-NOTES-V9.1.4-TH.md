# Tanjai AI Studio V9.1.4 — Prompt Brain Engine

เวอร์ชันนี้ยกระดับระบบจาก “ตัวต่อข้อความ Prompt” เป็น “ระบบช่วยคิดบรีฟแบบมืออาชีพ” สำหรับงานประชาสัมพันธ์ ภาพ โพสต์ วิดีโอ เสียงพากย์ สไลด์ และชุดสื่อ

## สิ่งที่เพิ่มใหม่

### 1) Prompt Brain Engine
- วิเคราะห์ประเภทงานจากหัวข้อ รายละเอียด หมวดงาน และบริบท
- แยก “ข้อมูลจริงที่ผู้ใช้ให้มา” ออกจาก “ข้อมูลที่ยังไม่ระบุ”
- ระบุชัดว่าอะไรห้าม AI แต่งเพิ่ม เช่น วันที่ เวลา สถานที่ ค่าสมัคร เบอร์ติดต่อ QR Code และโลโก้
- ให้คะแนนความพร้อมของบรีฟ
- สร้าง Creative Direction อัตโนมัติแบบปลอดภัย

### 2) Smart Creative Direction
ระบบช่วยเลือกแนวทางภาพ / โพสต์ / วิดีโอ ตามบริบท เช่น
- เดินวิ่งเพื่อการกุศล
- แจ้งข่าว / ประกาศ
- สรุปกิจกรรม / ลงพื้นที่
- ให้ความรู้ / ขั้นตอนบริการ
- ประชุมประชาคม
- ไว้อาลัย / ลดสี
- โปรโมทเพลง

### 3) Prompt แบบ “ลงมือทำทันที”
ปรับ Prompt ทุกเมนูให้สั่ง AI ปลายทางทำงานทันที ไม่ใช่ตอบกลับเป็นแค่สรุปบรีฟ

### 4) Safe Missing Data Handling
หากข้อมูลไม่ครบ ระบบจะสั่งให้ใช้คำว่า “จากข้อมูลเบื้องต้น” และเว้นช่องเติมภายหลัง แทนการแต่งข้อมูลจริงขึ้นเอง

### 5) Output Delivery Guard
ยังคง guard กัน AI ตอบกลับเป็น path ภายใน เช่น `/mnt/data/file.png` โดยไม่มีไฟล์จริงหรือลิงก์ดาวน์โหลด

### 6) Identity Lock
งานที่เกี่ยวข้องกับภาพบุคคลจริงยังยึดกฎเดิม:
- ห้ามเปลี่ยนใบหน้า
- ห้ามสลับหน้า
- ห้ามแก้โครงหน้า
- ห้าม identity drift
