# V9.3.9 — Balanced Cover + Lite Album Frame System

ฐาน: V9.3.8 Facebook Album Frame Fidelity Fix

ขอบเขต:
- แก้เฉพาะเมนู “ชุดภาพโพสต์ Facebook”
- เมนูอื่นคงเดิม

แนวทาง:
- ใช้ตัวอย่าง Cover Frame + Lite Album เป็นแนวทางโครงสร้าง ไม่ลอกขนาดกรอบให้ใหญ่เว่อร์
- กรอบต้องช่วยนำสายตา ไม่บังภาพจริง
- ข้อความต้องอ่านได้บนมือถือ แต่ไม่กินพื้นที่ภาพเกินจำเป็น
- Preview และไฟล์ที่ดาวน์โหลดต้องใช้ logic เดียวกัน

สิ่งที่แก้:
- ปรับ Cover Frame ให้เป็นกรอบพรีเมียมแบบสมดุล ลดความหนาของขอบและแถบข้อความจากรอบก่อน
- ปรับ Lite Frame ให้มีกรอบพอเห็นว่าเป็นชุดเดียวกัน แต่เบากว่า Cover
- ปรับ Additional Frame ให้บางและโปร่งกว่า Lite
- ลดความสูงแถบข้อความของ Cover / Lite / Additional
- ลดขนาด badge และโลโก้ในภาพรองเพื่อไม่บังภาพ
- ล้าง UI ลำดับภาพที่เบียดกัน ให้เหลือการ์ดสั้น ๆ: ปก / Lite / Additional
- ย้ายคำอธิบายยาวไปไว้ใน details “โครงอัลบั้ม”

ตรวจแล้ว:
- JavaScript syntax ผ่าน
- จำนวนไฟล์ต่ำกว่า 100
- Version badge ยังอยู่บนหน้า Login และ Topbar
