# V9.3.9 — Balanced Cover + Lite Album Frame System

ฐาน: V9.3.7 Facebook Album Smart Slot Layout

แก้เฉพาะระบบชุดภาพโพสต์ Facebook

สิ่งที่แก้:
- แก้ปัญหา "กรอบไม่ออกตามแบบ" ในภาพ Lite และ Additional
- เพิ่มการใช้ Pro Frame กับภาพรองด้วย ไม่ใช่เฉพาะภาพ Cover
- Lite Frame มีกรอบชัดขึ้น, ป้ายเลขมุมซ้ายบน, โลโก้มุมขวาบน, และแถบล่างแบบอัลบั้ม
- Additional Frame มีกลิ่นอายกรอบเดียวกัน แต่ยังคงเบากว่า Lite
- คง version badge บนหน้า Login และ Topbar

ผลลัพธ์:
- ภาพรอง 2–4 จะไม่ดูเหมือนภาพเปล่าใส่เส้นบางอีกต่อไป
- ภาพ 5+ มีกรอบบางแบบต่อเนื่องกับชุดหลัก
- ชุดอัลบั้มโดยรวมใกล้ reference frame มากขึ้น
