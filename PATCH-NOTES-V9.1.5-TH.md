# Tanjai AI Studio V9.1.5 — Professional Brief-to-Prompt Output

แพตช์นี้ต่อจาก V9.1.4 Home Card 8 โดยเน้นแก้ “ข้อความที่คัดลอกจากเว็บ” ให้ใช้งานจริงแบบมืออาชีพมากขึ้น

## สิ่งที่เพิ่ม / แก้ไข

### 1) เพิ่ม Professional Brief-to-Prompt Output Layer
- เพิ่มไฟล์ `js/pro-brief-output.js`
- ทำหน้าที่เป็นชั้นส่งออก Prompt มืออาชีพหลัง Prompt Brain Engine
- แยกข้อมูลเป็น 4 กลุ่มชัดเจน:
  - ข้อมูลจริงที่ผู้ใช้ให้มา
  - ข้อมูลที่ยังไม่ระบุและห้ามแต่งเอง
  - สิ่งที่ AI ช่วยคิดต่อได้อย่างปลอดภัย
  - ข้อห้าม / กติกาความถูกต้อง

### 2) ปรับ Prompt หลักให้เป็น “คำสั่งพร้อมใช้” จริง
- ผลลัพธ์หลักที่กดคัดลอกจะไม่ปนรายงานวิเคราะห์ยาว ๆ
- โครงสร้าง Prompt ใหม่มีหัวข้อชัดเจน:
  1. บทบาทของ AI
  2. งานที่ต้องทำ
  3. ข้อมูลจริง
  4. ข้อมูลที่ยังไม่ระบุ
  5. สิ่งที่ AI ช่วยคิดได้
  6. Creative Direction
  7. แนวทางมืออาชีพ
  8. ข้อห้าม
  9. รูปแบบผลลัพธ์

### 3) แยก Template ตามเมนูชัดขึ้น
- สร้างภาพ
- เรียบเรียงเนื้อหา / โพสต์
- งานพิธีกร
- วิดีโอ
- เสียงพากย์
- สไลด์
- สร้างชุดสื่อ

### 4) เพิ่ม Guard สำหรับงานภาพและวิดีโอ
- ห้ามตอบกลับเป็น path ภายใน เช่น `/mnt/data/xxx.png`
- หากสร้างภาพจริงไม่ได้ ต้องบอกเหตุผลและส่ง Prompt ภาพพร้อมใช้แทน
- หากมีภาพบุคคลจริง ต้องคงอัตลักษณ์เดิม 100%
- ห้าม facial reconstruction, face replacement, facial feature modification และ identity drift
- ห้ามสร้างโลโก้ปลอม / QR Code ปลอม / เบอร์โทรปลอม

### 5) ปรับ UX ปุ่มคัดลอก
- เปลี่ยนข้อความปุ่มเป็น “คัดลอก Prompt พร้อมใช้”
- ปุ่ม “คัดลอก + เปิด GPT” จะคัดลอกข้อความแล้วเปิด GPT ปลายทาง
