# ทันใจ AI Studio V9.3.14 — Free Writing Team

เวอร์ชันนี้ปรับเฉพาะงานเขียนและบท ไม่เปลี่ยนระบบสร้างภาพ ชุดภาพ Facebook หรือความสามารถด้านภาพเดิม

## ผลงานที่สร้างจบในเว็บ

- Caption & Article Writer: สรุปงาน บทความ ข่าว โพสต์ Facebook ข้อความ LINE และแคปชั่น
- MC Writer: สคริปต์พิธีกร คำเชื่อม ลำดับพิธี และบัตรเตือนหน้างาน
- Video Script Writer: Hook, Storyboard, บทพากย์ และข้อความบนจอ
- Voice Script Writer: สคริปต์เสียงพร้อมอ่าน จังหวะ คำเน้น และช่องตรวจคำอ่าน
- Slide Writer: เนื้อหาบนสไลด์พร้อม Speaker Notes ตามจำนวนหน้าที่เลือก

ทุกบทบาทอ่านข้อมูลผ่าน Content Brain ร่วมกัน และมี Fact Guard ตรวจชื่อ หน่วยงาน สถานที่ วันที่ ตัวเลข และข้อมูลที่ยังขาด โดยไม่แต่งข้อเท็จจริงใหม่

## โหมดฟรี

ระบบทำงานในเบราว์เซอร์โดยไม่ใช้ API และไม่ส่งข้อมูลออกไปภายนอก ผลลัพธ์หลักพร้อมคัดลอกใช้ได้ทันที ส่วน Prompt ขั้นสูงของวิดีโอ เสียง และสไลด์ยังเก็บเป็นทางเลือกในรายละเอียดเพิ่มเติม

ข้อจำกัด: ระบบฟรีไม่อ่านข้อความในภาพเอกสารอัตโนมัติ ผู้ใช้ต้องกรอกสาระสำคัญในช่องรายละเอียดเพื่อให้ Fact Guard ตรวจได้
