OPTION 1: GitHub Pages (Recommended - Free & Easy)
─────────────────────────────────────────────────
1. Create GitHub account (if you don't have one)
2. Create new repository: "tanjai-ai-studio"
3. Upload all files from this folder
4. Go to Settings → Pages
5. Select: Branch "main", Folder "/ (root)"
6. Wait 1-3 minutes
7. Your site: https://YOUR-USERNAME.github.io/tanjai-ai-studio

────────────────────────────────

2. Click "New site from Git"
3. Connect your GitHub repository

5. Your site: https://your-site-name.netlify.app

───────────────────────────────

🔍 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────

Problem: Files not loading
Solution: Hard refresh (Ctrl+F5), check browser console for 404 errors

Problem: Styling looks wrong
Solution: Clear browser cache, hard refresh

Problem: localStorage not working
Solution: Check browser privacy settings, not available in private mode

Problem: Voice not working
Solution: Not all browsers support Web Speech API, check browser console

Problem: Mobile layout broken
Solution: Verify viewport meta tag, check CSS media queries
