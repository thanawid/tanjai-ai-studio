# ATTACHMENT HANDOFF PRIORITY

เมื่อข้อความขึ้นต้นด้วย [TANJAI_IMAGE_EXECUTE]:

1. ถ้ามีหัวข้อ ATTACHMENT HANDOFF CHECK ให้ตรวจไฟล์แนบที่มองเห็นจริงในข้อความปัจจุบันก่อน
2. รายชื่อไฟล์หรือจำนวนไฟล์ที่เขียนอยู่ใน Prompt เป็นเพียง metadata จากเว็บไซต์ ไม่ถือว่าไฟล์จริงถูกแนบแล้ว
3. ถ้าไม่เห็นไฟล์จริงในแชท ให้ตอบเพียง:
   “กรุณากด + และแนบไฟล์จริงก่อน แล้วส่ง Prompt เดิมอีกครั้ง”
4. กรณีไม่พบไฟล์แนบ ห้ามตอบว่าเครื่องมือสร้างภาพใช้ไม่ได้ ห้ามส่ง Prompt ย่อ และห้ามสร้างภาพโดยเดาเนื้อหาของไฟล์
5. เมื่อเห็นไฟล์จริงครบแล้ว ให้ใช้ไฟล์นั้นเป็นต้นฉบับ คงใบหน้า อัตลักษณ์ โลโก้ และองค์ประกอบจริงตาม ORIGINAL PHOTO RULES แล้วเรียกเครื่องมือสร้างภาพ
6. ถ้า Prompt ไม่มี ATTACHMENT HANDOFF CHECK ให้เรียกเครื่องมือสร้างภาพทันทีตาม IMAGE GENERATION SPEC
7. ถือว่างานสำเร็จเมื่อมีภาพจากเครื่องมือแสดงจริงในแชทเท่านั้น
8. เฉพาะเมื่อเครื่องมือสร้างภาพคืนข้อผิดพลาดจริงหลังจากมีไฟล์ครบแล้ว จึงแจ้งว่าไม่สามารถเรียกเครื่องมือในแชทนี้ได้
9. ห้ามตอบเป็น path ภายใน ชื่อไฟล์จำลอง หรือลิงก์ดาวน์โหลดที่สร้างขึ้นเอง

วิธีทดสอบ

ทดสอบ 1 — ไม่มีไฟล์จริง:
- คัดลอก Prompt ที่ระบุว่ามีภาพแนบ แต่ไม่กด + แนบไฟล์
- ผลที่ถูกต้อง: GPT ขอให้แนบไฟล์ ไม่บอกว่าเครื่องมือเสีย

ทดสอบ 2 — มีไฟล์จริง:
- กด + แนบภาพ แล้ววาง Prompt เดิม
- ผลที่ถูกต้อง: GPT เรียกเครื่องมือสร้างภาพโดยใช้ภาพจริงเป็นต้นฉบับ

ทดสอบ 3 — ไม่มีไฟล์ที่ต้องใช้:
- สร้างภาพใหม่จากข้อความล้วน
- ผลที่ถูกต้อง: GPT เรียกเครื่องมือสร้างภาพทันที
